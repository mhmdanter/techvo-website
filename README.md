## 🚀 Deployment

This site is deployed via **cPanel Git Version Control** on Namecheap Stellar hosting.

### Auto-Deploy Workflow
1. Edit any file directly on GitHub
2. Commit the changes
3. Go to **cPanel → Git Version Control → Manage → Pull or Deploy**
4. Click **"Deploy HEAD Commit"**
5. ✅ Changes are live on **techvo.co** within seconds

### Manual Upload (alternative)
1. Download files as ZIP from GitHub
2. Upload to **cPanel → File Manager → public_html**
3. Extract and replace files
